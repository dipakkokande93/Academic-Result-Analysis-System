# 📊 Multi-Divisional Academic Analytics System

## 🎯 Project Goal
To automate the performance analysis of students across **18 divisions**, reducing manual reporting time by 90%.

## 💡 Key Solutions Provided:
- **Topper Logic:** Extracts Top 3 students per division and overall.
- **Subject Difficulty Index:** Identifies subjects with the lowest passing rates to alert faculty.
- **Division Benchmarking:** Compares all 18 divisions to find high-performing teaching patterns.

## 🛠️ Built With:
- **Python** (Pandas for data cleaning, NumPy for ranking logic)
